Create `src/lib/actions/recruitment.actions.ts`. Start with `"use server";` as the first line (mandatory project pattern).

Imports:
- `revalidatePath` from `next/cache`
- `auth` from `@/lib/auth`
- `prisma` from `@/lib/prisma`
- `fs` from `fs/promises`
- `path` from `path`

Add `requireHRAdmin()` helper (identical pattern to payroll.actions.ts — checks session, checks role is HR_ADMIN or SUPER_ADMIN, returns ServiceResult<{ userId: string }>).

Implement these exported async server actions. Each validates input with safeParse, returns early if auth fails, returns `{ success: false, error: string }` on validation failure.

**createVacancyAction(input: unknown): Promise<ServiceResult<{ vacancyId: string }>>**
- Validate with createVacancySchema.safeParse
- Create vacancy: `prisma.vacancy.create({ data: { title, departmentId, positionId, description, requirements, openDate: new Date(openDate), createdBy: userId } })`
- revalidatePath('/recruitment')
- Return `{ success: true, data: { vacancyId: vacancy.id } }`

**updateVacancyStatusAction(input: unknown): Promise<ServiceResult<null>>**
- Validate with updateVacancyStatusSchema.safeParse
- Update: `prisma.vacancy.update({ where: { id: vacancyId }, data: { status } })`
- revalidatePath('/recruitment')
- Return `{ success: true, data: null }`

**createCandidateAction(input: unknown): Promise<ServiceResult<{ candidateId: string }>>**
- Validate with createCandidateSchema.safeParse
- Create candidate: `prisma.candidate.create({ data: { vacancyId, namaLengkap, email: email || null, nomorHp: nomorHp || null } })`
- revalidatePath(`/recruitment/${vacancyId}`)
- Return `{ success: true, data: { candidateId: candidate.id } }`

**updateCandidateStageAction(candidateId: string, newStage: string): Promise<ServiceResult<null>>**
- Call requireHRAdmin() directly (no schema — called from Kanban drag handler with typed args)
- Find candidate, verify exists
- Update: `prisma.candidate.update({ where: { id: candidateId }, data: { stage: newStage as any } })`
- revalidatePath(`/recruitment/${candidate.vacancyId}`)
- Return `{ success: true, data: null }`

**scheduleInterviewAction(input: unknown): Promise<ServiceResult<null>>**
- Validate with scheduleInterviewSchema.safeParse
- Upsert interview (create or update — `@@unique` on candidateId): `prisma.interview.upsert({ where: { candidateId }, update: { scheduledAt: new Date(scheduledAt), interviewerId, notes: notes || null }, create: { candidateId, scheduledAt: new Date(scheduledAt), interviewerId, notes: notes || null } })`
- Also move candidate to INTERVIEW stage: `prisma.candidate.update({ where: { id: candidateId }, data: { stage: "INTERVIEW" } })`
- Run both in a transaction: `prisma.$transaction([...])`
- Find vacancy for revalidate: `prisma.candidate.findUnique({ where: { id: candidateId }, select: { vacancyId: true } })`
- revalidatePath(`/recruitment/${vacancyId}`)
- Return `{ success: true, data: null }`

**updateOfferFieldsAction(input: unknown): Promise<ServiceResult<null>>**
- Validate with updateOfferFieldsSchema.safeParse
- Update candidate: `{ offeredSalary: offeredSalary ? new Decimal(offeredSalary) : undefined, startDate: startDate ? new Date(startDate) : undefined, notes: notes || undefined }`
- Import `Decimal` from `decimal.js` for the conversion
- revalidatePath on candidate's vacancy path
- Return `{ success: true, data: null }`

**convertCandidateToEmployeeAction(candidateId: string, employeeId: string): Promise<ServiceResult<null>>**
- Called AFTER HR submits the employee creation form and has the new employeeId
- Check candidate exists and `!candidate.isConverted` — return error if already converted
- If candidate has cvFilePath: copy the CV file to `uploads/employees/${employeeId}/` using `fs.copyFile`. Source: `path.join(process.cwd(), candidate.cvFilePath)`. Destination dir: `path.join(process.cwd(), 'uploads', 'employees', employeeId)`. Call `fs.mkdir(destDir, { recursive: true })` first. Destination file: same filename. Create `EmployeeDocument` record: `{ employeeId, documentType: "LAINNYA", fileName: candidate.cvFileName!, filePath: relative path to copied file, fileSize: candidate.cvFileSize!, mimeType: candidate.cvMimeType! }`.
- Update candidate: `{ isConverted: true, convertedAt: new Date(), convertedToId: employeeId }`
- Run candidate update and (conditional) EmployeeDocument creation in a transaction
- revalidatePath on candidate's vacancy path
- Return `{ success: true, data: null }`
