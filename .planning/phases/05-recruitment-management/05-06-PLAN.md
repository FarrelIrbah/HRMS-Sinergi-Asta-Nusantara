```typescript

export interface OfferLetterData {
  candidateName: string;
  positionTitle: string;

  startDate: string;      // pre-formatted: "dd MMMM yyyy" in Bahasa Indonesia
```

StyleSheet: page padding 50, A4 size. Letter style:
- `header`: company name bold large centered at top
- `meta`: date and "Nomor:" line (use placeholder "—")
- `addressBlock`: "Kepada Yth." + candidate name
- `subject`: "Perihal: Surat Penawaran Kerja" underlined bold
- `body`: paragraph text 11pt, lineHeight 1.6
- `signature`: "Hormat kami," + company name + blank line + HR admin name

Formal Bahasa Indonesia letter body (3 paragraphs):
1. Salam pembuka dan perkenalan perusahaan, menyatakan bahwa kandidat dinyatakan diterima untuk posisi vacancyTitle di departemen departmentName.
2. Detail penawaran: gaji sebesar Rp {offeredSalary formatted with thousand separators}, dengan tanggal mulai kerja {startDate}. Harap konfirmasi penerimaan surat ini dalam 7 hari kerja.
3. Penutup: ucapan selamat dan terima kasih atas ketertarikan kandidat bergabung dengan perusahaan.

Format salary: use `offeredSalary.toLocaleString('id-ID')` — this works in Node.js.

GET handler:
- Auth: HR_ADMIN or SUPER_ADMIN only. Return 401/403 otherwise.
- Find candidate with vacancy (include department) and interview. Return 404 if not found.
- Guard: `if (candidate.stage !== "DITERIMA") return new Response("Forbidden", { status: 403 })`.
- Guard: `if (!candidate.offeredSalary || !candidate.startDate) return new Response("Gaji dan tanggal mulai belum diisi", { status: 400 })`.
- Get HR admin name from session: `session.user.name ?? "HR Admin"`.
- Format dates: `format(candidate.startDate, "dd MMMM yyyy", { locale: localeId })`, `format(new Date(), "dd MMMM yyyy", { locale: localeId })`.
- Build data: OfferLetterData object.
- Generate PDF using exact pattern from payslip route: `React.createElement(OfferLetterDocument, { data }) as ReactElement<DocumentProps, ...>`, `renderToStream`, chunk loop, Buffer.concat.
- Return Response with headers `Content-Type: application/pdf` and `Content-Disposition: attachment; filename="surat-penawaran-${candidate.namaLengkap.replace(/\s+/g, '-')}.pdf"`.
