**src/app/(dashboard)/recruitment/[vacancyId]/candidates/[candidateId]/_components/cv-upload.tsx** — "use client":
- Props: `{ candidateId: string }`.
- Renders a file input (`<input type="file" accept=".pdf,.jpg,.jpeg,.png">`) and an "Upload CV" button.
- On file select: validate file size ≤ 5MB and type is pdf/jpeg/png. If invalid, show error.
- On submit: `const formData = new FormData(); formData.append('file', file)`. POST to `/api/recruitment/candidates/${candidateId}/cv`. On success response: `toast.success("CV berhasil diupload")`, `router.refresh()`. On error: `toast.error(errorMessage)`.

**src/app/api/recruitment/candidates/[id]/cv/route.ts** — API route, mirror of `src/app/api/employees/[id]/documents/route.ts`:
- Auth: HR_ADMIN or SUPER_ADMIN only. Return 401/403 otherwise.
- Accept POST FormData with `file` field.
- Validate: file exists, size ≤ 5MB (5 * 1024 * 1024), type is `application/pdf`, `image/jpeg`, or `image/png`.
- Find candidate: `prisma.candidate.findUnique({ where: { id } })`. Return 404 if not found.
- Write to disk: `uploads/candidates/${id}/${Date.now()}-${file.name}`. Use `fs.mkdir({ recursive: true })` first. Write with `fs.writeFile`.
- Update candidate: `prisma.candidate.update({ where: { id }, data: { cvFileName: file.name, cvFilePath: `uploads/candidates/${id}/${filename}`, cvMimeType: file.type, cvFileSize: file.size } })`.
- Return 200 `{ success: true }`.
- Import `fs/promises` and `path` from Node stdlib. Import prisma from `@/lib/prisma`. Import auth from `@/lib/auth`.
