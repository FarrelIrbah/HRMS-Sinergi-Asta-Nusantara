**src/app/(dashboard)/recruitment/[vacancyId]/_components/kanban-board.tsx** — "use client":

```typescript
"use client";
import {
  DndContext, DragEndEvent, DragOverlay, DragStartEvent,
  KeyboardSensor, PointerSensor, closestCorners,
  useSensor, useSensors, type UniqueIdentifier,
} from "@dnd-kit/core";
import {
  SortableContext, sortableKeyboardCoordinates,
} from "@dnd-kit/sortable";
import { useState } from "react";
import { toast } from "sonner";
import { updateCandidateStageAction } from "@/lib/actions/recruitment.actions";
import { CandidateStage } from "@/types/enums";
import { KanbanColumn } from "./kanban-column";
import { CandidateCard } from "./candidate-card";
```

State: `candidatesByStage` — a Record<CandidateStage, Candidate[]> initialized from props. `activeCandidate` — the candidate being dragged (for DragOverlay).

STAGES constant (in display order): `["MELAMAR", "SELEKSI_BERKAS", "INTERVIEW", "PENAWARAN", "DITERIMA", "DITOLAK"]`

`onDragStart(event: DragStartEvent)`: find the candidate by event.active.id across all stages, set activeCandidate. Save `previousState` snapshot: `setPreviousState(structuredClone(candidatesByStage))`.

`onDragEnd(event: DragEndEvent)`:
- Get `over` container. If no `over`, reset activeCandidate and return.
- Find source stage (which column the card came from) by checking which column's array contains event.active.id.
- Target stage = the `over.id` (column id) OR the column that contains `over.id` if `over.id` is a card id (find by scanning columns).
- If source === target, reset and return.
- Optimistic update: remove candidate from source column array, add to target column array.
- Reset activeCandidate.
- Call server action: `updateCandidateStageAction(candidateId, targetStage)`. On failure: revert state to previousState, `toast.error("Gagal memperbarui stage kandidat")`.

Column labels map: `{ MELAMAR: "Melamar", SELEKSI_BERKAS: "Seleksi Berkas", INTERVIEW: "Interview", PENAWARAN: "Penawaran", DITERIMA: "Diterima", DITOLAK: "Ditolak" }`

Render: horizontal scrollable container. For each stage, render KanbanColumn with SortableContext. DragOverlay shows CandidateCard with `isOverlay` prop.
