Create `src/lib/services/payroll.service.ts`. This service orchestrates the batch payroll run.

Key logic:

1. `runPayroll(month: number, year: number)` — main batch function:
   - Fetch all active employees with: baseSalary, allowances (isFixed), ptkpStatus, npwp, joinDate, and their attendance records for the given month (for overtime minutes and absence count)
   - For each employee:
     a. Compute `bpjsBasis` = baseSalary + sum(fixed allowances)
     b. Compute overtimePay: overtimeMinutes × (baseSalary / 173 / 60) using Decimal, where 173 = monthly norm hours per Indonesian law
     c. Compute absenceDeduction: unpaidAbsenceDays × (baseSalary / 22) using Decimal, where unpaid leave = approved LeaveRequests with isPaid=false in that month, plus AttendanceRecords with status ABSENT (no clock-in, no approved leave)
     d. Compute grossPay = bpjsBasis + overtimePay − absenceDeduction
     e. Call `calculateBPJS(new Decimal(bpjsBasis))` — BPJS basis is fixed salary only, not overtime
     f. Call `calculateMonthlyPPh21(grossPay, ptkpStatus)` for month 1-11; for month 12, call `calculateDecemberPPh21` with annualized data fetched from prior months' PayrollEntries
     g. Compute totalDeductions = bpjs.totalEmployeeDeduction + pph21
     h. Compute netPay = grossPay − totalDeductions
   - Find or create a PayrollRun for (month, year) — if it exists and is FINALIZED, throw error "Payroll sudah difinalisasi"
   - If DRAFT exists, delete all existing PayrollEntries for this run before re-inserting (re-runnable)
   - Upsert PayrollRun (DRAFT status)
   - Create all PayrollEntry rows in a single `prisma.payrollEntry.createMany()`
   - Return the PayrollRun with entry count

2. `finalizePayroll(payrollRunId: string)`:
   - Fetch the PayrollRun, confirm it is DRAFT
   - Update status to FINALIZED
   - Return updated run

3. `getPayrollRuns()` — return all PayrollRuns ordered by year desc, month desc, with entry count

4. `getPayrollRunDetail(payrollRunId: string)` — return PayrollRun with all PayrollEntries including employeeName, totals

**December special handling** for calculateDecemberPPh21:
```typescript
// For December, fetch all prior PayrollEntries (Jan-Nov) for this employee in this year
const priorEntries = await prisma.payrollEntry.findMany({
  where: {
    employeeId: employee.id,
    payrollRun: { year, month: { lt: 12 } }
  },
  include: { payrollRun: true }
});
const annualGross = priorEntries.reduce((sum, e) => sum.plus(new Decimal(e.grossPay.toString())), new Decimal(0)).plus(grossPay);
const annualBpjsEmp = priorEntries.reduce(
  (sum, e) => sum.plus(new Decimal(e.bpjsJhtEmp.toString())).plus(new Decimal(e.bpjsJpEmp.toString())),
  new Decimal(0)
).plus(bpjs.jhtEmp).plus(bpjs.jpEmp);
const totalPPh21JanNov = priorEntries.reduce((sum, e) => sum.plus(new Decimal(e.pph21.toString())), new Decimal(0));
```

**Important:** All Decimal values from Prisma come as `Prisma.Decimal` — convert with `new Decimal(prismaDecimal.toString())` before arithmetic.

**src/lib/actions/payroll.actions.ts** — create server actions:
```typescript
"use server";
import { requireHRAdmin } from "@/lib/auth-utils"; // same pattern as employee.actions.ts
import { runPayrollSchema, finalizePayrollSchema, updateEmployeeSalarySchema } from "@/lib/validations/payroll";
import { runPayroll, finalizePayroll } from "@/lib/services/payroll.service";
import prisma from "@/lib/prisma";
import { revalidatePath } from "next/cache";

export async function runPayrollAction(input: unknown) {
  await requireHRAdmin();
  const data = runPayrollSchema.parse(input);
  const result = await runPayroll(data.month, data.year);
  revalidatePath("/payroll");
  return { success: true, payrollRunId: result.id, entryCount: result._count?.entries ?? 0 };
}

export async function finalizePayrollAction(input: unknown) {
  await requireHRAdmin();
  const data = finalizePayrollSchema.parse(input);
  await finalizePayroll(data.payrollRunId);
  revalidatePath("/payroll");
  return { success: true };
}

export async function updateEmployeeSalaryAction(input: unknown) {
  await requireHRAdmin();
  const data = updateEmployeeSalarySchema.parse(input);

  // Update baseSalary on employee
  await prisma.employee.update({
    where: { id: data.employeeId },
    data: { baseSalary: data.baseSalary },
  });

  // Replace allowances: delete existing, insert new
  if (data.allowances !== undefined) {
    await prisma.employeeAllowance.deleteMany({ where: { employeeId: data.employeeId } });
    if (data.allowances.length > 0) {
      await prisma.employeeAllowance.createMany({
        data: data.allowances.map((a) => ({
          employeeId: data.employeeId,
          name: a.name,
          amount: a.amount,
          isFixed: a.isFixed,
        })),
      });
    }
  }

  revalidatePath(`/employees/${data.employeeId}`);
  return { success: true };
}
```
