- alamat (String?)
- nomorHp (String?)

- departmentId (String)
- positionId (String)
- officeLocationId (String?) -- optional, for attendance later
- contractType (ContractType)
- joinDate (DateTime)
- isActive (Boolean @default(true))
- terminationDate (DateTime?)
- terminationReason (String?)
- npwp (String?)
- ptkpStatus (PTKPStatus?)
- bpjsKesehatanNo (String?)
- bpjsKetenagakerjaanNo (String?)
- createdAt, updatedAt
- Relations: user (User), department (Department), position (Position), documents (EmployeeDocument[]), emergencyContacts (EmergencyContact[])
- @@map("employees")

Add model EmployeeDocument:
- id, employeeId, documentType (DocumentType), fileName, filePath, fileSize (Int), mimeType, createdAt, updatedAt
- Relation: employee (Employee, onDelete: Cascade)
- @@map("employee_documents")

Add model EmergencyContact:
- id, employeeId, name, relationship, phone, address (String?), createdAt, updatedAt
- Relation: employee (Employee, onDelete: Cascade)
- @@map("emergency_contacts")

Update existing models:
- User: add `employee Employee?` relation
- Department: add `employees Employee[]` relation
- Position: add `employees Employee[]` relation
